# Documentação do projeto

Bem-vindo à documentação inicial do projeto.

Este repositório está em fase de preparação e será implementado com arquitetura **MVC (Model-View-Controller)**.

## Objetivo desta documentação

- Definir a base arquitetural antes do início da implementação.
- Registrar padrões para manter consistência técnica.
- Facilitar onboarding e alinhamento do time.

## Por onde começar

1. Leia a visão geral do projeto.
2. Revise a arquitetura MVC e o fluxo de requisição.
3. Consulte a estrutura de pastas proposta.
4. Siga os padrões e convenções definidos.
5. Acompanhe o roadmap inicial do MVP.
