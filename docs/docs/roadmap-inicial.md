# Roadmap inicial

## Fase 1 — Fundação

- Definir stack e dependências principais.
- Criar estrutura de pastas MVC.
- Configurar ambiente local e variáveis.

## Fase 2 — Primeiros fluxos

- Implementar autenticação básica (se aplicável).
- Criar primeiro módulo com CRUD mínimo.
- Documentar endpoints e decisões técnicas.

## Fase 3 — Qualidade

- Adicionar testes unitários para regras de negócio.
- Adicionar testes de integração dos fluxos principais.
- Configurar lint e validação em CI.

## Fase 4 — Preparação para produção

- Revisar segurança básica e tratamento de erros.
- Definir estratégia de logs e monitoramento.
- Planejar versionamento e release.

## Critérios de pronto (MVP)

- Fluxo principal funcional de ponta a ponta.
- Arquitetura MVC aplicada com separação clara.
- Documentação mínima atualizada para onboarding.
