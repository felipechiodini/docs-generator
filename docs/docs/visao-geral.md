# Visão geral do projeto

Este projeto ainda está em fase inicial e será desenvolvido seguindo a arquitetura **MVC (Model-View-Controller)**.

## Objetivo

Construir uma aplicação organizada, escalável e de fácil manutenção, separando responsabilidades entre:

- **Model**: regras de negócio e acesso a dados.
- **View**: apresentação dos dados para o usuário.
- **Controller**: orquestração das requisições e fluxo da aplicação.

## Status atual

- Projeto **não iniciado**.
- Esta documentação define a base de estrutura e padrões.

## Escopo inicial (MVP)

- Estrutura base de pastas MVC.
- Roteamento inicial.
- Primeiros módulos de domínio.
- Páginas essenciais para validação do fluxo.

## Próximos passos

1. Definir stack técnica definitiva.
2. Criar esqueleto MVC.
3. Implementar primeiro caso de uso ponta a ponta.
4. Configurar testes e pipeline de qualidade.
